Born Robot - Song Generator

  • Version 1 has a very limited skill level, developing the song from a seed idea using fairly random (perlin noise) changes to create a simple melody and rythm.
  • Version 2 Send MIDI signals out of the MIDI interface.
  • Version 3 (in development) Record audio. Save MIDI and audio.
  • Version 4 (not started) a completely different model. Uses all 12 notes of a chromatic scale, weighted to diatonic notes. The melody is driven by a set of chord progressions
